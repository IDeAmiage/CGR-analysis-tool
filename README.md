### Installation 
Download git bash (64-bit version):
https://git-scm.com/download/win

- open git bash
- clone this repository:  ```git clone <url>```

### Download dependencies:
```shell
cd CGR-analysis-tool
pip install poetry
poetry install
```

### Run
Run .py file:
```shell
poetry run python scripts/example_py.py 
```
